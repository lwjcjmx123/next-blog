# 智能单词学习浏览器插件

智能单词学习浏览器插件是一款专为英语学习者设计的 Chrome 扩展程序。它能够在用户浏览网页时提供即时的单词翻译服务，并将学习记录自动保存到 Notion 数据库中，帮助用户建立个人化的单词学习体系。

## 🚀 核心功能

### 划词翻译

- 🖱️ **即时翻译**：选中网页中的单词即可获得翻译
- 🎯 **精准识别**：智能识别单词边界，支持复合词翻译
- ⚡ **快速响应**：毫秒级翻译响应，不影响阅读体验

### AI 智能翻译

- 🤖 **DeepSeek AI 集成**：采用先进的 AI 模型提供准确翻译
- 🌐 **多语言支持**：支持英中、中英双向翻译
- 📖 **上下文理解**：结合语境提供更准确的翻译结果

### Notion 数据库集成

- 📝 **自动保存**：翻译的单词自动保存到个人 Notion 数据库
- 📊 **学习统计**：记录学习进度和频率统计
- 🔄 **数据同步**：支持多设备间的学习数据同步

### 个性化配置

- ⚙️ **API 配置**：支持自定义 Notion API 和 AI 服务配置
- 🎨 **界面定制**：可调整弹窗样式和显示偏好
- 🔧 **模型选择**：支持切换不同的 AI 翻译模型

## 🛠️ 技术架构

### 前端技术栈

- **React 18**：现代化的用户界面框架
- **TypeScript**：提供完整的类型安全保障
- **Vite**：快速的构建工具和开发服务器
- **CSS3**：现代化的样式设计

### 扩展程序特性

- **Manifest V3**：符合最新 Chrome 扩展标准
- **Content Scripts**：注入页面脚本实现划词功能
- **Background Service Worker**：后台服务处理 API 请求
- **Storage API**：本地存储用户配置和缓存

### 外部服务集成

- **Notion API**：数据持久化和同步
- **DeepSeek AI API**：智能翻译服务
- **Chrome Extension APIs**：浏览器扩展功能

## 📱 使用体验

### 安装和配置

1. 从 Chrome Web Store 安装扩展程序
2. 配置 Notion API 密钥和数据库 ID
3. 设置 AI 翻译服务参数
4. 开始享受智能学习体验

### 日常使用

1. **划词翻译**：在任意网页选中单词查看翻译
2. **保存学习**：点击保存按钮将单词添加到学习库
3. **复习回顾**：在 Notion 中查看和复习学习记录
4. **进度跟踪**：通过统计数据了解学习进展
