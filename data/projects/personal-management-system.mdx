# 个人管理系统

个人管理系统是一个现代化的生产力管理工具，旨在帮助用户高效管理日常任务、时间和资源。该系统采用 Next.js 14 构建，支持多种存储模式，为不同需求的用户提供灵活的解决方案。

## 🚀 核心功能

### 📋 任务管理

- **智能任务分类**：支持项目分组、优先级设置、截止日期管理
- **进度追踪**：可视化任务完成进度，支持子任务分解
- **标签系统**：灵活的标签管理，便于任务分类和检索

### 📅 日历系统

- **多视图支持**：月视图、周视图、日视图切换
- **事件管理**：支持重复事件、提醒设置
- **时间块规划**：可视化时间分配，提高时间利用效率

### 📝 笔记功能

- **富文本编辑**：支持 Markdown 语法，代码高亮
- **分类管理**：笔记本分组，标签关联
- **全文搜索**：快速定位所需内容

### 💰 财务追踪

- **收支记录**：详细的收入支出管理
- **分类统计**：按类别、时间维度分析财务状况
- **图表展示**：直观的数据可视化

## 🛠️ 技术架构

### 前端技术栈

- **Next.js 14**：最新的 React 全栈框架，支持 App Router
- **TypeScript**：类型安全的 JavaScript 超集
- **Tailwind CSS**：实用优先的 CSS 框架
- **Framer Motion**：流畅的动画效果

### 后端与数据

**IndexedDB 模式**

- 完全本地化存储
- 无需服务器部署
- 离线优先设计
- 数据隐私保护

### 🎨 现代化 UI/UX

- **响应式设计**：完美适配桌面和移动设备
- **暗黑模式**：护眼的深色主题
- **流畅动画**：基于 Framer Motion 的交互动效
- **直观操作**：符合用户习惯的交互设计

### ⚡ 性能优化

- **服务端渲染**：Next.js SSR 提升首屏加载速度
- **代码分割**：按需加载，减少包体积
- **缓存策略**：智能缓存机制，提升用户体验
- **PWA 支持**：渐进式 Web 应用特性

## 🚀 部署方案

### Vercel 一键部署

系统针对 Vercel 平台进行了优化：

```bash
# 克隆项目
git clone https://github.com/lwjcjmx123/dashboard-1.git
cd dashboard-1

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```
