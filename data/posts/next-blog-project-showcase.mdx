# Next.js 个人博客项目

这是一个基于 Next.js 15 构建的现代化个人博客系统，旨在为开发者提供一个完整的个人展示平台。

## 项目简介

本项目是一个全栈的个人网站解决方案，集成了博客系统、项目展示和在线简历三大核心功能。采用现代化的技术栈，提供优秀的用户体验和开发体验。

## 主要功能

### 📝 博客系统

- 支持 MDX 格式文章编写
- 文章分类和标签管理
- 搜索和筛选功能
- 响应式阅读体验

### 🚀 项目展示

- 项目详情页面
- 技术栈标签
- 项目链接和源码
- 图片展示支持

### 👤 在线简历

- 个人信息展示
- 技能和经验
- 教育背景
- 联系方式

### 🎨 用户体验

- 深色/浅色主题切换
- 响应式设计
- 快速加载
- SEO 优化

## 技术栈

### 前端技术

- **Next.js 15**: React 全栈框架，支持 SSR/SSG
- **TypeScript**: 类型安全的 JavaScript
- **Tailwind CSS**: 原子化 CSS 框架
- **Framer Motion**: 动画库

### 开发工具

- **MDX**: 支持 React 组件的 Markdown
- **Apollo Client**: GraphQL 客户端
- **next-themes**: 主题切换
- **Radix UI**: 无障碍 UI 组件

### 部署平台

- **Vercel**: 主要部署平台
- **Cloudflare**: 备用部署方案

## 项目结构

```
app/
├── blog/           # 博客相关页面
├── projects/       # 项目展示页面
├── resume/         # 简历页面
└── layout.tsx      # 全局布局

components/
├── ui/             # 基础 UI 组件
├── providers/      # Context 提供者
└── ...             # 业务组件

data/
├── posts/          # 博客文章 (MDX)
├── projects/       # 项目信息
└── resume/         # 简历数据

lib/
├── data.ts         # 数据获取函数
├── utils.ts        # 工具函数
└── apollo-client.ts # GraphQL 配置
```

## 快速开始

### 📦 安装依赖

```bash
npm install
```

### 🚀 启动开发服务器

```bash
npm run dev
```

### 🏗️ 构建生产版本

```bash
npm run build
npm start
```

### 📝 添加内容

1. 在 `data/posts/` 目录下创建 `.mdx` 文件
2. 在 `data/projects/` 目录下添加项目信息
3. 修改 `data/resume/resume.json` 更新简历

## 自定义配置

### 🎨 主题定制

- 修改 `tailwind.config.js` 调整颜色和样式
- 编辑 `app/globals.css` 自定义全局样式

### 📊 数据配置

- `lib/data.ts`: 数据获取逻辑
- `data/` 目录: 内容数据存储

### 🔧 功能配置

- `next.config.js`: Next.js 配置
- `package.json`: 依赖和脚本管理
